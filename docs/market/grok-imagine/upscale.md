        ## Task ID Source


        The `task_id` parameter should be obtained from a previously generated
        video task. You can upscale videos that were created using:


        <Card title="Generate videos First" icon="lucide-webhook"
        href="/market/grok-imagine/text-to-video">
          Create videos using the text-to-video model to get a task ID for upscaling.

        **How to get a task ID:**

        1. Use the [Text to video API](/market/grok-imagine/text-to-video) to
        generate videos.

        2. Extract the `taskId` from the API response.

        3. Use that `taskId` as the `task_id` parameter in the upscale request.


        :::info[Information]

        Only videos generated by **Kie AI models** can be upscaled. External
        videos are not supported.

        :::

        </Card>


        :::tip[Pro Tip]

        For production use, we recommend using the `callBackUrl` parameter to
        receive automatic notifications when upscaling completes, rather than
        polling the status endpoint.

        :::
